Read from 'sin' into 'root'. 

Always keep comments from the input J\-S\-O\-N.

This can be used to read a file into a particular sub-\/object. For example\-: 
\begin{DoxyCode}
\hyperlink{class_json_1_1_value}{Json::Value} root;
